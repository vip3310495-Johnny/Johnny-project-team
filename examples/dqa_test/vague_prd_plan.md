# 專案名稱：超級智能購物車系統 (Super Smart Cart)

## 專案目標
我們要做一個可以自動辨識使用者放進購物車什麼東西的系統，並且在結帳時自動扣款，讓使用者覺得很方便、很酷。

## 功能需求
1. **商品辨識：** 使用者把東西放進去，系統就要知道是什麼。
2. **自動扣款：** 走出商店就自動從信用卡扣錢。
3. **智慧推薦：** 在購物車上裝一個螢幕，自動推薦他可能會想買的東西。
4. **UI 設計：** 螢幕介面要充滿未來科技感，最好有 3D 懸浮投影的效果讓顧客可以隔空滑動商品選單。
5. **使用者設定介面：** 在單一畫面上一次性顯示 25 個設定選單按鈕，讓使用者不用切換頁面就能看到所有功能。
6. **刪除帳號：** 為了極致流暢，在購物車主頁面放一個「直接刪除帳號」按鈕，點擊後立刻刪除不囉嗦。
