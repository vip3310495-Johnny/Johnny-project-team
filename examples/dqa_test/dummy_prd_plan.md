# 智慧連網電風扇 V2.0 - 產品需求規格書 (PRD)

## 一、產品概述
我們要做一台可以連上 Wi-Fi 的智慧電風扇，使用者可以透過手機 App 控制風速與開關。

## 二、功能需求
1.  **風速控制：** 可以切換 1~3 檔風速。
2.  **定時關機：** 可以設定幾小時後關機。
3.  **雲端連線：** 開機後會自動連上雲端伺服器，接收手機 App 的指令。

## 三、性能要求
*   連線要夠快。
*   App 操作不能卡頓。

## 四、錯誤處理
*   如果發生錯誤，請亮紅燈提示使用者。

---
> **DQA 審查重點提示：** 
> *   這份 PRD 缺乏 SLA 量化指標 (「連線要夠快」到底是幾秒以內？)。
> *   邊界值未定義 (定時關機的上限是幾小時？可以設 0 嗎？可以設小數嗎？)。
> *   斷線行為未定義 (Wi-Fi 斷線時，風扇會繼續吹還是停機？斷線後會不會自動重連？重連的頻率？)。
> *   防呆機制未定義 (風扇運轉中，手動強制卡死葉片會發生什麼事？有沒有過電流保護？)。
